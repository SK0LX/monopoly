# monopoly
Монополия - игра


Данная игра - монополия, которую можно развернуть на собственном сервере

Стек - C#, postgreSQL

## Текущий набросок

Пока база данных не подключена: данные лежат в in-memory репозиториях и пропадают после перезапуска.

Запуск API:

```bash
dotnet run --project app/Monopoly.csproj
```

Пока оставлена минимальная точка входа:

- `GET /` - проверка, что API запущен
